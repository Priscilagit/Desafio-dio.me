Como usar o Git e fazer um upload para o Github.

    Usar uma pasta contida no arquivo C do seu computador.
  • Cd(nome da pasta que deseja entrar)+/.
  • Mkdir: para adicionar uma outra pasta dentro dessa já existente.
  • Ls: mostra as documentos contidos dentro dessa pasta.
  • Ls -a: mostra os documentos ocultos dessa pasta.
  • Git init: para inicializar o git dentro dessa pasta
  • Git config --global user.email ("colocar email usado no Github") configurar email antes de fazer o arquivo.
  • Git config --global user.name "colocar nome usado no Github"
  • Git add . : para adicionar o que foi mudado no arquivo:
  • Git commit -m "nome do commit" : e o objeto que da o significado as alterações.
