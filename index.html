Como usar o Git e fazer um upload para o Github.

    Usar uma pasta contida no arquivo C do seu computador.
  • Cd(nome da pasta que deseja entrar)+/.
  • Mkdir: para adicionar uma outra pasta dentro dessa já existente.
  • Ls: mostra as documentos contidos dentro dessa pasta.
  • Ls -a: mostra os documentos ocultos dessa pasta.
  • Git init: para inicializar o git dentro dessa pasta
  • Git config --global user.email ("colocar email usado no Github") configurar email antes de fazer o arquivo.
  • Git config --global user.name "colocar nome usado no Github"
  • Git add . : para adicionar o que foi mudado no arquivo:
  • Git commit -m "nome do commit" : e o objeto que da o significado as alterações.
  • Git status: mostra se tem alteração no arquivo.
  • Mv "nome do arquivo": para mover um arquivo.
  • echo > : cria uma arquivo README.md
  • Git config --list : mostra as configuraçãoes que você fez nesse arquivo.

    • Trabalhando no Github

  • Criar uma conta no Github e usar o mesmo email que você configurou no Git.
  • Criar um repositório para depois colocar o arquivo que você criou.
  • Retorna ao Git
      • No git ....
  • Git remote add origin: " adicionar a url que foi criado no repositório".
  • Git remote -v : mostra os arquivos que vai ser colocado nesse repositório.
  • Git push -u origin main : puxa o arquivo da sua maquina para dentro do repositório criado.
  • Git pull origin : é usado para empurar o arquivo quando não consegui com o Git push.
  • Retornar ao Github
      • No Github...
  • Se tudo der certo, lá estará o arquivo criado e lá você verá todos os commits que você fez.
